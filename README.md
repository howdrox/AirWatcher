# How to run

The default app
```
make && target/airwatcher
```

The unit tests
```
make test && target/test_airwatcher
```

# Dependencies

## GoogleTest linux install
```
sudo apt install libgtest-dev
cd /usr/src/gtest
sudo cmake CMakeLists.txt
sudo make
sudo cp lib/*.a /usr/lib
```
